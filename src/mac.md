## Custom defaults

### Autohide dock time

```sh
// 0.2 seconds
defaults write com.apple.dock autohide-time-modifier -float 0.2; killall Dock;
// Restore defaults
defaults delete com.apple.dock autohide-time-modifier; killall Dock;
```
